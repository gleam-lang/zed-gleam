# Changelog

## v0.6.0

- Add comment language injections to allow styling of `TODO`, `FIXME`, etc. comments when the `Comments Highlighter` extension is installed
- Use more specific capture names for functions to enable themes to use separate colors for function definitions and function calls
- Update to the latest Tree-sitter grammar

## v0.5.2

- Update to the latest Tree-sitter grammar
- Remove Hexdocs provider, as the `/docs` slash command was removed from Zed

## v0.5.1

- Add support for `////` module doc comments

## v0.5.0

- Update to the latest Tree-sitter grammar

## v0.4.0

- Highlight `echo` as a keyword
- Fix downloading Gleam on Windows by using `.zip` extension, as Windows distributions use a zipfile rather than a tarball

## v0.3.0

- Add brackets queries to support the "move to enclosing bracket" editor action
- Update to the latest Tree-sitter grammar

## v0.2.1

- Initial release
